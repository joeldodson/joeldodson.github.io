Relevance to Python

  • Okay, Python isn't very popular for building UIs
  • Though, with frameworks like Django, Flask, and simlar others, there's plenty of HTML being served probably via rendering templates
    • And there are existing packages to build UIs in Python, making those UIs accessible is a struggle at best

Cool New Options for 'building UIs in Python

  • BeeWare - cross platform desktop and mobile apps using native components
  • Flet - also cross platform desktop and mobile based on Flutter
  • PyScript - running Python in the browser using WASM, maybe can use it in Electron or Tauri eventually as well

The Future has Potential

  • For BeeWare, Using native components is a great start toward achieving accessibility
  • For PyScript, accessibility is fairly well understood in the browser (there are issues, that's a different talk)
  • Frameworks can encapsulate many details of accessibility. Developers will still need to understand the ideas though and be aware of what is and isn't provided by the frmework
