Analysis of Udemy for BIT A11y Course, 2023-08

Udemy is a SPA (Single Page Application), or at least parts of it are. This results in URLs being a bit odd in places. For example, to get to the signup page, I clicked a button to open a side panel. Then I clicked on the signup button. As you can see below, this resulted in query parameters being inserted in the URL.

Update, 2023-09-20, FWIW - the signup now appears to be a link on the main page, the side panel seems to be gone. The link I follow is still the same for signup, including all the query parameters. Maybe they got rid of the side panel.

Failures for Operable

Operable Failure 1

  1. Page Title: Online Courses - Learn Anything, On Your Schedule | Udemy
  2. Page URL: https://www.udemy.com
  3. Success Criteria - Failed: Heading and Labels
  4. Success Criteria Identifier: 2.4.6
  5. Description: This page contains many short descriptions of courses each at an h3 heading. The end of each course description has a button with label, "Show course details."Nothing in the label associates the button with the course name.

    I initially had this as a 2.4.4 Link Purpose violation as the label is missing context. I realized 2.4.4 is specific to links. 2.4.6 is not well defined in WCAG 2.1 itself but has slightly more in the understanding section indicating lack of clarity could be a violation.

  6. Assistive Technologies Used: NVDA Screen Reader
  7. Date of Analysis: 2023-09-20

Operable Failure 2

  1. Page Title: Online Courses - Learn Anything, On Your Schedule | Udemy
  2. Page URL: https://www.udemy.com
  3. Success Criteria - Failed: Keyboard
  4. Success Criteria Identifier: 2.1.1
  5. Description: This page contains many short descriptions of courses each at an h3 heading. The end of each course description has a button with label, "Show course details." with no indication of whether the button will expand a section or initiate a popup.

    I press the space bar and hear "expanded." I arrow up and down but cannot find any additional information for the course.

    Using the Chrome developer tools, I inspect the element. It is a <button> element and has a class name indicating it will result in some kind of popup window. I didn't find any indication though of additional information anywhere.

  6. Assistive Technologies Used: NVDA Screen Reader
  7. Date of Analysis: 2023-09-20

Operable Failure 3

  1. Page Title: Udemy
  2. Page URL: https://support.udemy.com/hc/en-us
  3. Success Criteria - Failed: Page Titled
  4. Success Criteria Identifier: 2.4.2
  5. Description: As you might infer from the URL, this is Udemys' support page. You'd never know that from the title. This page is linked to from udemy.com (and many other pages), it's not a one off zombie page I found lurking in the far off internets of yesteryear.
  6. Assistive Technologies Used: NVDA Screen Reader
  7. Date of Analysis: 2023-09-20

Operable Failure 4

  1. Page Title: Online Courses - Learn Anything, On Your Schedule | Udemy
  2. Page URL: https://www.udemy.com
  3. Success Criteria - Failed: Link Purpose (In Context)
  4. Success Criteria Identifier: 2.4.4
  5. Description: Under an h2 heading for subscription plans for individuals and organizations, there are 2 links labeled, "view plan" and "view plans". The purpose of these links is not clear when viewing a list of links on the site, or when navigating via "next link".

    "View subscription plan for individuals" and "view subscription plans for organizations", respectively, would be better names for these links.

  6. Assistive Technologies Used: NVDA Screen Reader
  7. Date of Analysis: 2023-09-21

Operable Failure 5

  1. Page Title: Online Courses - Learn Anything, On Your Schedule | Udemy
  2. Page URL: https://www.udemy.com/courses/search/?src=ukw&q=playwright
  3. Success Criteria - Failed: Bypass Blocks
  4. Success Criteria Identifier: 2.4.1
  5. Description: "Skip to content" on the search results page directs to the Udemy home page.

    According to NVDA (using nvdakey+k), the skip to content link on the search results page is, https://www.udemy.com/courses/search/#main-content-anchor. Pressing enter on that link takes me back to the Udemy home page.

    The search results page has an h1 heading which includes the number of results. My hope/expectation is the skip to content would take me to that h1.

  6. Assistive Technologies Used: NVDA Screen Reader
  7. Date of Analysis: 2023-09-21
